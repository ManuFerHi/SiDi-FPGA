# Laser500_MiST

Video Technology Laser 350/500/700 for the MiST / MiSTica / SiDi FPGA by Antonino Porcino (Nippur72)

Usage - Put *.rbf and ROM into your SD.

Source - https://github.com/nippur72/Laser500_MiST
